# Biografía - Paul Abreu Beriguete

Página HTML profesional que contiene la biografía completa de Paul Abreu Beriguete, incluyendo datos personales, formación académica, experiencia profesional y horario de clases.

## 📋 Contenido

- **Datos Personales**: Información básica y de contacto
- **Contexto Familiar**: Antecedentes familiares y entorno
- **Formación Académica**: Educación recibida y estudios universitarios
- **Trayectoria Profesional**: Experiencia laboral y cargos desempeñados
- **Horario de Clases**: Tabla con materias, horas y aulas

## 🎨 Características

✅ Diseño moderno y responsivo  
✅ Gradientes de color profesionales  
✅ Foto de perfil  
✅ Tabla de horario interactivo  
✅ Timeline de experiencia laboral  
✅ Compatible con dispositivos móviles  

## 📁 Estructura del Proyecto

```
biography-portfolio/
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── foto.jpg            # Foto de perfil
└── README.md           # Este archivo
```

## 🚀 Cómo Usar

1. Clona el repositorio
2. Sube tu foto con el nombre `foto.jpg` en la raíz del proyecto
3. Abre `index.html` en tu navegador

## 📸 Instalación de la Foto

Reemplaza el archivo `foto.jpg` con tu foto personal. El archivo debe estar en la raíz del proyecto junto con `index.html` y `styles.css`.
